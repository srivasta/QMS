Detailed Description

A concrete, example server.

This is a concrete implementation of a QMS Sensor. This is supposed to be the reference implementation of a sensor that does not have any domain specific information whatsoever, and should serve as a basis for other Sensor implementations.

This implementation includes Local resource discovery, publishing capabilities to enable global resource discovery, implementation of QoS aware adaptive.
