Send the test messages for the probe.

Use the stored information to send messages to the probe in the following order

  1. Send first name and last name.
  2. Send pause action message.
  3. Send kill action message , which is a wrong command.
  4. Send restart action message, which is a wrong command.
  5. Send resume action message.
  6. Send stop action message.
