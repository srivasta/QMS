Detailed Description

Every probe that has a Sensor make an offer on its behalf in the QMS Trader must have a pointer to the XML Schema that it is using to represent its data. This should be available to the Sensor as well as the clients that wish to communicate with it, thus facilitating the validation of the XML data that is created. Additional information needed for the sensor to figure out how to handle the probe is included probeSchema - the URI to the schema used by the probe schemaVersion - version of the schema
