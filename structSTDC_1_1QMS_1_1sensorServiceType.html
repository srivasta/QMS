Detailed Description

Each Sensor offers many services to the QMS world and each of those services will make offers into the QMS Trader. Since each Sensor implements the Command and Control Corba interface, at a minimum, there will be a Sensor service that will use the mandatoryCorbaOfferProperties in the choice below. This is the case with the Generic Sensor. However, if the QMS citizen (Sensor) offers up other services from built-in probes, for instance, then it makes the corresponding offers in the QMS Trader again.
