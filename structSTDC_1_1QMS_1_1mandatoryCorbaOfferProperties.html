Detailed Description

If the entity making the offer also provides a CORBA interface (all QMS Citizens do and other clients may as well) we'd like to identify the IDL interfaces they offer up to the world with specific version numbers. This will permit clients to figure out if they are using the right version of the IDL interface while trying to talk to the CORBA entity.
