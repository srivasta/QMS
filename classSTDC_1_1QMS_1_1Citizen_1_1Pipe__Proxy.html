Detailed Description

A pure virtual class to interact with probes using simple pipes.

This is the root of a heirarchy of classes that provides a polymorphic abstraction layer for interacting with probes using a simple pipe (standard UNIX paradigm).

This adds the knowledge of the pathname of the executable to run as a child, the command line argument that one provides the child process, and the environment that the child process runs in.
