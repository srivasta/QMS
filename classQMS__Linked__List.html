Detailed Description

template<class NodeType, class KeyType, class Lock>
class QMS_Linked_List< NodeType, KeyType, Lock >

This class implements a double linked list.

This is one of the concrete container classes in the multilevel data structure framework. This implements most of the common stl like members.
