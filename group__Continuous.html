Detailed Description

The Continuous Probe is capable of providing the Metrics in a continuous manner.

This consists of two threads -> 1) Worker thread and 2) Listener thread.

Listener Thread:- The Listener thread listens for commands on the standard input and passes that information to the Worker thread using an enum structure.

Worker Thread:- The Worker thread calls the get_data function at the specified intervals. This thread is resonsible for actions like Starting,Pausing, Resuming and Stopping the Probe.

Primary functions of Continuos Probe are

  1. Supports QMS Comands like Start, Pause ,Resume and Stop.
  2. Continuously monitors the standard input for commands mentioned above.
  3. Converts the Metrics information to XML format.
  4. Continuously prints the Metrics information in the form of XML document on the standard output at the given interval.
