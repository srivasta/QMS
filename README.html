This file serves as the starting point for everyone who wishes to understand how to build, install and run QMS. QMS is under continuous development at System/Technology Development Corporation. To obtain the latest version of QMS, please visit the S/TDC web site at http://www.stdc.com/QMS.

The documentation for QMS is distributed throughout the source code. This file serves as a ready guide to help you find the various pieces of interest. Note that to get a thorough insight into QMS and its operation, we recommend that you do build the QMS documentation as described in Installing QMS.

The files described below are included in the Documentation subdirectory and serve as a good starting point to beginning with QMS.

Contents 

- @ref README.background
  - This file gives you the background necessary to understand
  the various pieces of QMS and the terminology in it.

- @ref README.install 
  - This file includes documentation on the system requirements
  to use QMS and the procedure to build QMS and install it. 

- @ref README.run
  - This file describes the steps to start deploying Sensors and
  Probes in a QMS framework.
