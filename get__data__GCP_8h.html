Detailed Description

Implementation of a one shot probe for Remos data.

This file defines the data structures and the method invoked to get data from the Remos system. This file is for a continuous probe – This is a more sophisticated probe that operates continuously, maintaining state in between data pulses. This probe implements its own periodic behavior, and only requires a rate setting to periodically pump out data.
