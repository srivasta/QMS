Detailed Description

template<class NodeType, class KeyType, class Lock>
class QMS_MLD_Containers< NodeType, KeyType, Lock >

This is the abstract superclass of Multi level containers.

This class provides the common API for most of the containers in the multilevel data structure framework. A multilevel data structure is, for example, can be like this:

  • The top level is a double linked list,
  • each node of the doubly linked list is the rot of a red black tree
  • each node of the red black tree is the contains a stack and so on. The implementation is provided by the concrete containers, and the iterators are also only provided by the concrete classes.
