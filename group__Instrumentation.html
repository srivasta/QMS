Detailed Description

By definition, Instrumentation is an integral part of QMS. QMS provides a mechanism for data providers to publish their presence on the grid, and provides data consumers to subscribe to, and even request tailored data from, such consumers.

QMS Instrumentation is implemented as a Sensor/Probe design pattern.

QMS provides a standard, flexible interface for both the interactions between the data providers and consumers, as well as a set of standard XML formats for encapsulation of the data generated.

By crafting standard QMS sensors that deploy or interface with the underlying probes, QMS provides standard sets of capabilities often not present in the underlying probes themselves.
