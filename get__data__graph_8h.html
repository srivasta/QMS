Detailed Description

Contains implementation of Remos Continuous Graph Probe.

The Remos interface is a uniform way for applications to monitor a network and the hosts connected to it. The Remos papers are available from the Remulac web page: http://www.cs.cmu.edu/~cmcl/remulac/index.html.

This sensor inserts a probe into the Remos system with the query parameters as requested. Additionally, this sensor provides a facility to periodically monitor the network (and replay the original query as requested).
