Detailed Description

template<class NodeType, class KeyType, class Lock>
class QMS_Linked_List_Iterator< NodeType, KeyType, Lock >

An iterator for a double linked list.

Iterate through the double-linked list. This class provides an interface that let users access the internal element addresses directly.
