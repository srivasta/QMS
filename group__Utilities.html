Detailed Description

These are modules that provide software ustilities and reuseable objects that can be used in other QMS components, or can be used by client software. A number of template libraries and reference implementations of various patterns are planned.

  • Debugging Tools
  • Benchmarking Tools
  • Test Harnesses
  • Utility Libraries
    • XML Parsing
    • File Backend (read/write)
    • Data Structures and other patterns
