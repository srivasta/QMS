Detailed Description

A pure virtual superclass for proxies to probes.

This is the root of a heirarchy of classes that provides a polymorphic abstraction layer for interacting with probes. For each distinct Sensor-Probe interaction protocol there shall be an derived class to encapsulate the unique interaction. This class sets up the (few) utility methods common to all proebes
