Detailed Description

ECInfo class definition and declaration of it's methods.

This class is used to store the EventChannel name and event id associated with a specific QMSUniqueName category. Each QMSUniqueName category is associated with three classes of event channels based on the class of the event being published or consumed. The event channel classes are Announce, Data or Request.

This class has accessor and mutator methods for accessing and mutating the values of EventChannel name and event id.

This class has a friend member function which is used for overloading << operator , so that "cout" object can be used to print the contents of this class.
