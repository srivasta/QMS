Detailed Description

template<class NodeType, class KeyType, class Lock>
class QMS_MLD_Factory< NodeType, KeyType, Lock >

This class is an superclass for various concrete factories.

This class serves as a base for concrete factories. A template factory, though attractive in that it would not require a concrete factory class for every multilevel container class implemented, turned out to be quite ugly, since factory objects are member objects of the very container objects they create.
