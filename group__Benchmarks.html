Collaboration diagram for Benchmarks - Non production code for performance measurement:

These are non production programs use to generate performance metrics for the QMS system. Based on the class ACE_High_Res_Timer, A high resolution timer class wrapper that encapsulates OS-specific high-resolution timers, such as those found on Solaris, AIX, Win32/Pentium, and VxWorks.
